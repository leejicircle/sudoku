# 🖌 클리어 결과 모달 (Clear Modal)

> 퍼즐 완성 시 표시되는 축하 모달. 결과 요약 + 액션 버튼.

## 2. 오버레이 (Backdrop)

| 속성 | 값 |
|------|-----|
| 배경 | `oklch(0 0 0 / 0.5)` (반투명 검정) |
| z-index | `--z-modal-overlay` (60) |
| 위치 | fixed, inset: 0 |
| 클릭 동작 | 모달 닫지 않음 (결과 확인 강제) |
| 진입 | fade-in, `--duration-slow` (300ms) |
| backdrop-filter | blur(4px) |

## 3. 모달 컨테이너

| 속성 | 값 |
|------|-----|

| 최대 높이 | 90vh |

| 모서리 | `--radius-xl` (14px) |

## 4. 콘텐츠 상세

### 4.1 축하 아이콘 영역

| 속성 | 값 |
|------|-----|
| 아이콘 | 커스텀 축하 아이콘 또는 `PartyPopper` (Lucide), 48px |
| 색상 | `--success` |
| 하단 간격 | 16px (`--space-4`) |
| 애니메이션 | bounce-in (scale 0→1.2→1, 500ms, `--ease-bounce`) |

### 4.2 제목

| 속성 | 값 |
|------|-----|
| 텍스트 | "퍼즐을 완성했어요!" |
| 폰트 | 24px (`--text-heading`), weight 700 |
| 색상 | `--foreground` |
| 정렬 | text-center |
| 하단 간격 | 24px (`--space-6`) |

### 4.3 결과 카드

```
┌──────────────────────────────────┐
│  📊 난이도       보통            │
│  ⏱  시간         03:24           │
│  💡 힌트         1회 사용        │
│  ⭐ 평가         ⭐⭐⭐          │
└──────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 배경 | `--muted` |
| 모서리 | `--radius-lg` (10px) |
| 패딩 | 20px (`--space-5`) |
| 너비 | 100% |
| 하단 간격 | 24px (`--space-6`) |
| 행 간격 | 12px (`--space-3`) |

### 4.4 결과 항목별 명세

| 항목 | 좌측 (라벨) | 우측 (값) |
|------|-----------|----------|
| **난이도** | `--muted-foreground`, 14px, "난이도" | `--difficulty-*` 색상, 14px bold |
| **시간** | `--muted-foreground`, 14px, "시간" | `--foreground`, `--font-geist-mono`, 16px bold |
| **힌트** | `--muted-foreground`, 14px, "힌트" | `--foreground`, 14px, "N회 사용" |
| **평가** | `--muted-foreground`, 14px, "평가" | `Star` 아이콘, 20px |

### 4.5 별점 표시

| 별점 | 조건 | 시각적 표현 |
|------|------|-----------|
| ⭐⭐⭐ | 힌트 0회 + 기준 시간 이내 | 별 3개 `--warning` (채움) |
| ⭐⭐ | 힌트 1~2회 또는 시간 초과 | 별 2개 채움 + 1개 `--muted` (빈) |
| ⭐ | 힌트 3회 이상 | 별 1개 채움 + 2개 빈 |

**별 애니메이션:** 순차적 scale-in (0→1), 각 100ms 딜레이

```
⭐ (0ms) → ⭐ (100ms) → ⭐ (200ms)
```

### 4.6 난이도별 기준 시간

| 난이도 | 3별 기준 시간 |
|--------|-------------|
| 쉬움 | 5분 이내 |
| 보통 | 10분 이내 |
| 어려움 | 20분 이내 |
| 전문가 | 40분 이내 |

## 5. 액션 버튼

### 5.1 레이아웃

| 속성 | 값 |
|------|-----|
| 배치 | flex-col, gap: 10px (`--space-2` + 2) |
| 너비 | 100% |
| 버튼 순서 | 다음 스테이지 → 랭킹 보기 → 홈으로 |

### 5.2 버튼별 명세

| 버튼 | 스타일 | 높이 | 아이콘 | 텍스트 |
|------|--------|------|--------|--------|
| 다음 스테이지 | **Primary** | 48px | `Play` (16px) | "다음 스테이지" |
| 랭킹 보기 | **Secondary** | 44px | `Trophy` (16px) | "랭킹 보기" |
| 홈으로 | **Ghost** | 44px | `Home` (16px) | "홈으로" |

### 5.3 "다음 스테이지" 버튼 로직

| 상황 | 동작 |
|------|------|
| 같은 난이도 계속 | 새 퍼즐 생성 → 게임 화면 |
| 새 난이도 해금 | 해금 토스트 표시 → 새 퍼즐 생성 |
| 최고 난이도 클리어 | "다음 스테이지" → "한 번 더" 로 텍스트 변경 |

### 5.4 랭킹 등록 분기

| 상태 | 동작 |
|------|------|
| 로그인 상태 | 자동 랭킹 등록 → "랭킹 보기" 시 내 순위 하이라이트 |
| 비로그인 | "랭킹 보기" 버튼 → "로그인하고 기록 저장" 으로 변경 |

## 6. 새 난이도 해금 알림

클리어로 인해 다음 난이도가 해금될 때 추가 표시:

```
┌──────────────────────────────────┐
│  🔓 새로운 난이도가 해금되었어요!  │
│     어려움 (Hard) 도전 가능       │
└──────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 위치 | 결과 카드와 버튼 사이 |
| 배경 | `--success` / 10% opacity |
| 보더 | 1px solid `--success` / 30% |
| 모서리 | `--radius-md` (8px) |
| 패딩 | 12px 16px |
| 아이콘 | `Unlock` (Lucide), 18px, `--success` |
| 제목 | 14px, weight 600, `--foreground` |
| 설명 | 13px, `--muted-foreground` |
| 진입 | fadeIn + slideDown, 300ms (결과 카드 후 400ms 딜레이) |

## 7. 애니메이션 타임라인

전체 클리어 모달의 애니메이션 순서:

```
0ms     │ 보드 완성 감지
0ms     │ board-complete-celebrate (보드 scale up, 500ms)
300ms   │ 오버레이 fade-in 시작 (300ms)
500ms   │ 모달 slide-up + fade-in 시작 (300ms)
800ms   │ 축하 아이콘 bounce-in (500ms, ease-bounce)
900ms   │ 제목 fade-in (200ms)
1000ms  │ 결과 카드 fade-in (200ms)
1100ms  │ 별점 순차 scale-in (별 1→2→3, 각 100ms)
1400ms  │ 해금 알림 slide-down (있을 경우, 300ms)
1500ms  │ 버튼 영역 fade-in (200ms)
```

> 총 ~1.7초. 사용자가 결과를 자연스럽게 읽을 수 있는 속도.

## 8. 반응형

| 브레이크포인트 | 변경사항 |
|---------------|---------|
| < 375px | 모달 너비: calc(100vw - 32px), 패딩 축소 24px 16px |
| 375px ~ 767px | 기본 (위 와이어프레임) |
| ≥ 768px | 모달 max-width: 420px, 패딩 40px 32px |
| ≥ 1024px | 동일 (모달은 항상 중앙 고정) |

## 9. 접근성

| 속성 | 값 |
|------|-----|
| role | dialog |
| aria-modal | true |
| aria-label | "퍼즐 완성 결과" |
| 포커스 트랩 | 모달 내부에서만 Tab 순환 |
