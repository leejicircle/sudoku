> 스도쿠 퍼즐을 푸는 핵심 화면. 보드, 도구바, 숫자패드로 구성.

## 2. Header (게임 전용)

| 영역 | 내용 | 스타일 |
|------|------|--------|
| 좌측 | ← 뒤로가기 | `ChevronLeft` 24px, 44×44px 터치 영역 |
| 중앙 | 난이도 라벨 + 타이머 | 난이도: 14px caption, `--difficulty-*` 색상 |
| | | 타이머: 20px (`--text-timer`), `--font-geist-mono`, weight 500 |
| 우측 | ⏸ 일시정지 | `Pause` 20px, 44×44px 터치 영역 |

### 2.1 타이머 포맷

| 경과 시간 | 표시 |
|----------|------|
| < 1시간 | `MM:SS` (예: 03:24) |
| ≥ 1시간 | `H:MM:SS` (예: 1:03:24) |

## 3. 게임 보드

### 3.1 보드 전체 명세

| 속성 | 값 |
|------|-----|
| 셀 크기 | `--cell-size` = 40px (모바일) |
| 셀 간 선 | `--board-gap-thin` = 1px, 색상: `--board-border-thin` |
| 3×3 박스 선 | `--board-gap-thick` = 2px, 색상: `--board-border` |
| 외곽선 | `--board-border-width` = 3px, 색상: `--board-border` |
| 보드 배경 | `--board-bg` |
| 보드 모서리 | `--radius-lg` (10px) — 외곽만 |
| 보드 그림자 | `--shadow-board` |
| 전체 너비 | (40×9)+(1×6)+(2×2)+(3×2) = 376px |
| 보드 외부 패딩 | `--board-padding` = 16px (좌우 대칭) |
| 정렬 | 수평 중앙 |

### 3.2 셀 내부 구조

```
┌──────────────────┐
│                  │  ← 40×40px
│        5         │  ← 숫자: 중앙 정렬
│                  │
└──────────────────┘

┌──────────────────┐
│  1   2   3       │  ← 메모 모드: 3×3 그리드
│  4       6       │     각 메모 영역: ~12px
│          9       │     --text-cell-memo (10px)
└──────────────────┘
```

### 3.3 셀 숫자 스타일

| 유형 | 폰트 | 크기 | 가중치 | 색상 |
|------|------|------|--------|------|
| 유저 입력 | `--font-geist-mono` | `--text-cell` (24px) | 500 | `--cell-default-foreground` |
| 초기 제공 | `--font-geist-mono` | `--text-cell-given` (24px) | 700 | `--cell-given-foreground` |
| 메모 | `--font-geist-mono` | `--text-cell-memo` (10px) | 400 | `--muted-foreground` |
| 에러 | `--font-geist-mono` | `--text-cell` (24px) | 500 | `--cell-error-foreground` |

## 4. 도구바 (Toolbar)

### 4.1 레이아웃

```
┌───────────┬───────────┬───────────┬───────────┐
│  ↩        │  ✕        │  ✏        │  💡       │
│ 되돌리기   │  지우기   │  메모     │ 힌트(3)   │
└───────────┴───────────┴───────────┴───────────┘
```

### 4.2 도구 버튼 명세

| 속성 | 값 |
|------|-----|
| 레이아웃 | flex, 4등분 (flex-1 × 4), gap: 8px |
| 버튼 크기 | flex-1 × 48px 높이 |
| 아이콘 크기 | 20px |
| 라벨 폰트 | 11px, `--text-small` 이하, weight 400 |
| 아이콘-라벨 간격 | 2px |
| 배경 | transparent |
| 텍스트 | `--muted-foreground` |
| 호버 | `--accent` 배경 |
| 활성(메모 ON) | `--sudoku-primary` 아이콘 + 텍스트, `--sudoku-primary`/10% 배경 |
| 비활성 | opacity: 0.4, pointer-events: none |
| 모서리 | `--radius-md` (8px) |

### 4.3 각 도구 상세

| 도구 | 아이콘 | 라벨 | 비활성 조건 |
|------|--------|------|------------|
| 되돌리기 | `Undo2` | "되돌리기" | 히스토리 없을 때 |
| 지우기 | `Eraser` | "지우기" | 선택 셀이 빈 셀 또는 초기값일 때 |
| 메모 | `PenLine` | "메모" | 없음 (항상 활성, 토글) |
| 힌트 | `Lightbulb` | "힌트(N)" | 잔여 힌트 0일 때 |

## 5. 숫자패드 (Number Pad)

### 5.1 레이아웃

```
┌──────┬──────┬──────┬──────┬──────┐
│  1   │  2   │  3   │  4   │  5   │
├──────┼──────┼──────┼──────┼──────┤
│  6   │  7   │  8   │  9   │      │
└──────┴──────┴──────┴──────┘      │
                                    ← 마지막 칸 비움 또는 삭제 버튼
```

**대안 배치 (권장 — 5열 1~2행):**

```
[1] [2] [3] [4] [5]
[6] [7] [8] [9] [⌫]
```

### 5.2 버튼 명세

| 속성 | 값 |
|------|-----|
| 레이아웃 | grid, 5열, gap: `--numpad-gap` (8px) |
| 버튼 크기 | `--numpad-button-size` (48px) × 48px |
| 배경 | `--card` |
| 텍스트 | `--foreground`, `--text-numpad` (24px), `--font-geist-mono`, weight 500 |
| 모서리 | `--radius-md` (8px) |
| 그림자 | `--shadow-numpad` |
| 호버 | `--accent` 배경 |
| 활성(누름) | `animate-numpad-press`, `--sudoku-primary` 배경, `--sudoku-primary-foreground` 텍스트 |
| 완료(9개 배치됨) | opacity: 0.3, pointer-events: none, 텍스트에 취소선 |

### 5.3 숫자 완료 표시

해당 숫자가 보드에 9개 모두 배치되면:
- 버튼 opacity: 0.3
- 배경: `--muted`
- 텍스트: `--muted-foreground`, 취소선 없이 흐리게

### 5.4 숫자패드 하단 여백

| 속성 | 값 |
|------|-----|
| 패드 하단 여백 | `env(safe-area-inset-bottom)` + 16px |
| 목적 | 홈 인디케이터 영역 확보 (iPhone 등) |

## 7. 인터랙션 피드백

### 7.1 셀 선택 시

| 효과 | 대상 | 스타일 |
|------|------|--------|
| 선택 셀 | 탭한 셀 | 배경: `--cell-selected`, 그림자: `--shadow-cell-selected` |
| 같은 행/열/박스 | 관련 20셀 | 배경: `--cell-highlighted` |
| 같은 숫자 | 동일 숫자 셀 | 배경: `--cell-same-number` |
| 트랜지션 | 모든 하이라이트 | `--duration-fast` (100ms), ease-out |

### 7.2 숫자 입력 시

| 상황 | 애니메이션 |
|------|-----------|
| 유효한 숫자 | `animate-cell-pop` (scale 0.85→1.05→1, 200ms) |
| 무효한 숫자 | `animate-cell-error-shake` (X축 흔들림, 300ms) |
| 행/열/박스 완성 | `animate-cell-complete-flash` (초록 플래시, 500ms) |
| 전체 완성 | `board-complete-celebrate` (전체 scale up, 500ms) → 클리어 모달 |

### 7.3 숫자패드 피드백

| 효과 | 스타일 |
|------|--------|
| 눌림 | `animate-numpad-press` (scale 0.92→1, 100ms) |
| 햅틱 | `navigator.vibrate(10)` (지원 기기) |
