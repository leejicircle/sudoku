# 🖌 잠금 칸 팝오버 (Lock Popover)

> 잠금 상태 셀을 탭했을 때 표시되는 안내 팝오버.

---

## 1. 와이어프레임

```
                ┌──────────────────────┐
                │  🔒 잠금된 칸입니다   │
                │                      │
                │  이전 난이도를 클리어  │
                │  하면 해금됩니다.     │
                │                      │
                │  필요: 보통 1회 클리어 │
                │                      │
                │       [ 확인 ]       │
                └──────────┬───────────┘
                           │  ← 말풍선 꼬리 (화살표)
                    ┌──────┴──────┐
                    │  🔒 (셀)    │
                    └─────────────┘
```

---

## 2. 팝오버 스타일

### 2.1 컨테이너

| 속성 | 값 |
|------|-----|
| 너비 | 240px |
| 최대 높이 | auto |
| 배경 | `--popover` |
| 텍스트 | `--popover-foreground` |
| 보더 | 1px solid `--border` |
| 모서리 | `--radius-xl` (14px) |
| 그림자 | `--shadow-lg` |
| 패딩 | 16px (`--space-4`) |
| z-index | `--z-popover` (50) |

### 2.2 위치 (Positioning)

| 속성 | 값 |
|------|-----|
| 기준 | 탭한 셀의 중앙 기준 |
| 기본 위치 | 셀 위쪽 (top) |
| 공간 부족 시 | 셀 아래쪽 (bottom) 으로 자동 전환 |
| 오프셋 | 셀에서 8px 떨어짐 |
| 꼬리 (arrow) | 6px, 셀 중앙을 가리킴, 같은 배경+보더 |
| 화면 밖 방지 | 좌우 최소 8px 여백 유지 (viewport clamp) |

## 3. 콘텐츠 구조

### 3.1 레이아웃

```
┌──────────────────────────────────┐
│  🔒  잠금된 칸입니다              │  ← 제목 행
│                                  │
│  이전 난이도를 클리어하면          │  ← 설명
│  해금됩니다.                      │
│                                  │
│  ┌──────────────────────────┐    │  ← 해금 조건 박스
│  │ 📋 보통 난이도 1회 클리어 │    │
│  └──────────────────────────┘    │
│                                  │
│           [ 확인 ]               │  ← CTA 버튼
└──────────────────────────────────┘
```

### 3.2 요소별 명세

| 요소 | 스타일 |
|------|--------|
| 잠금 아이콘 | `Lock` (Lucide), 18px, `--muted-foreground` |
| 제목 | "잠금된 칸입니다", 16px (`--text-body`), weight 600, `--foreground` |
| 아이콘-제목 간격 | 8px (inline-flex) |
| 아이콘-제목 행 하단 간격 | 8px |
| 설명 | 14px (`--text-caption`), weight 400, `--muted-foreground` |
| 설명 하단 간격 | 12px |
| 해금 조건 박스 배경 | `--muted` |
| 해금 조건 박스 모서리 | `--radius-md` (8px) |
| 해금 조건 박스 패딩 | 10px 12px |
| 해금 조건 아이콘 | `ClipboardCheck` (Lucide), 14px |
| 해금 조건 텍스트 | 13px, weight 500, `--foreground` |
| 해금 조건 박스 하단 간격 | 12px |
| 확인 버튼 | Secondary 스타일, 36px 높이, full width |
| 확인 버튼 텍스트 | "확인", 14px, weight 500 |

## 4. 난이도별 해금 조건 텍스트

| 잠금 난이도 | 해금 조건 텍스트 |
|------------|-----------------|
| 보통 (Medium) | "쉬움 난이도 1회 클리어" |
| 어려움 (Hard) | "보통 난이도 1회 클리어" |
| 전문가 (Expert) | "어려움 난이도 1회 클리어" |

> 쉬움(Easy)은 잠금 없음이므로 팝오버 미표시.

## 5. 애니메이션

### 5.1 진입 애니메이션

```css
@keyframes popover-enter {
  0% {
    opacity: 0;
    transform: scale(0.96) translateY(4px);
  }
  100% {
    opacity: 1;
    transform: scale(1) translateY(0);
  }
}
```

| 속성 | 값 |
|------|-----|
| 애니메이션 | `popover-enter` |
| Duration | `--duration-normal` (200ms) |
| Easing | `--ease-out` |
| transform-origin | 꼬리(arrow) 방향 기준 |

### 5.2 퇴장 애니메이션

```css
@keyframes popover-exit {
  0% {
    opacity: 1;
    transform: scale(1);
  }
  100% {
    opacity: 0;
    transform: scale(0.96);
  }
}
```

| 속성 | 값 |
|------|-----|
| Duration | `--duration-fast` (100ms) |
| Easing | `--ease-in` |

## 6. 닫기 동작

| 트리거 | 동작 |
|--------|------|
| "확인" 버튼 | 팝오버 닫기 |
| 팝오버 바깥 영역 탭 | 팝오버 닫기 |
| 다른 셀 탭 | 팝오버 닫기 + 새 셀 선택 |
| ESC 키 | 팝오버 닫기 |
| 스크롤 | 팝오버 닫기 |

---

## 7. 홈 화면 잠금 팝오버 (변형)

홈 화면의 잠금된 난이도 카드를 탭했을 때도 유사한 팝오버를 사용하되, 위치와 크기가 다름:

| 속성 | 게임 보드 셀 | 홈 난이도 카드 |
|------|-------------|---------------|
| 너비 | 240px | 280px |
| 위치 | 셀 위/아래 | 카드 위/아래 |
| 추가 CTA | "확인" | "확인" + "이전 난이도 도전 →" |

---

## 8. 접근성

| 속성 | 값 |
|------|-----|
| role | `dialog` |
| aria-label | "잠금 칸 안내" |
| aria-modal | false (non-modal popover) |
| 포커스 | 팝오버 열릴 때 "확인" 버튼으로 포커스 이동 |
| 포커스 트랩 | 없음 (non-modal) |
| ESC | 닫기 후 트리거 셀로 포커스 복귀 |
