# 🖌 로그인 페이지

> OAuth 기반 소셜 로그인 페이지. 간결한 중앙 정렬 레이아웃.

---

## 1. 와이어프레임 (375px 모바일)

```
┌─────────────────────────────────────┐  ← 0px
│  Header (56px)                      │
│  ← 뒤로         "로그인"            │
├─────────────────────────────────────┤  ← 56px
│                                     │
│                                     │
│                                     │
│              🧩                     │  ← 앱 로고 (64px)
│                                     │
│           스도쿠                     │  ← 앱 이름 (text-heading, 24px)
│   스도쿠에 오신 것을 환영합니다       │  ← 인사말 (text-caption, 14px)
│                                     │
│                                     │  ← 간격 40px
│  ┌─────────────────────────────┐    │
│  │  🔵 Google로 계속하기       │    │  ← Google 버튼
│  └─────────────────────────────┘    │
│                                     │  ← 간격 12px
│  ┌─────────────────────────────┐    │
│  │  🟢 네이버로 계속하기       │    │  ← Naver 버튼
│  └─────────────────────────────┘    │
│                                     │
│                                     │  ← 간격 24px
│   게임은 로그인 없이도 즐길 수       │  ← 캡션 (text-small, 12px)
│   있어요                            │
│                                     │
│                                     │
│  (BottomNav 없음)                   │
└─────────────────────────────────────┘
```

---

## 2. 레이아웃 명세

| 속성 | 값 |
|------|-----|
| 전체 구조 | flex column, items-center, justify-center |
| 콘텐츠 영역 | min-h: calc(100vh - 56px), padding: 40px 24px |
| 콘텐츠 max-width | 320px (버튼 영역) |
| BottomNav | 숨김 (로그인 페이지에서는 불필요) |

## 3. 로고 & 텍스트 영역

| 요소 | 스타일 |
|------|--------|
| 앱 로고 | 64px × 64px, 스도쿠 그리드 아이콘 또는 `Grid3X3` (Lucide) |
| 로고 색상 | `--sudoku-primary` |

| 인사말 하단 간격 | 40px (`--space-10`) |
| 텍스트 정렬 | text-center |

## 4. OAuth 버튼 컴포넌트

### 4.1 공통 스타일

| 속성 | 값 |
|------|-----|
| 너비 | 100% (max-width: 320px) |
| 높이 | 48px (터치 타겟 준수) |
| 모서리 | `--radius-md` (8px) |
| 폰트 | 16px (`--text-body`), weight 500 |
| 아이콘 크기 | 20px (좌측) |
| 아이콘-텍스트 간격 | 12px |
| 정렬 | flex, items-center, justify-center |
| 트랜지션 | background `--duration-fast` (100ms) |
| 버튼 간 간격 | 12px (`--space-3`) |

### 4.2 Google 버튼

| 속성 | 값 |
|------|-----|
| 배경 | `oklch(0.98 0 0)` (거의 흰색) |
| 보더 | 1px solid `--border` |
| 텍스트 | `oklch(0.30 0 0)` (진한 회색) |
| 텍스트 내용 | "Google로 계속하기" |
| 아이콘 | Google 로고 SVG (공식 멀티컬러) |
| 호버 | 배경 `oklch(0.96 0 0)` |
| 활성 | 배경 `oklch(0.94 0 0)`, scale(0.98) |
| 그림자 | `--shadow-sm` |

### 4.3 Naver 버튼

| 속성 | 값 |
|------|-----|
| 배경 | `oklch(0.62 0.17 145)` (네이버 초록 #03C75A) |
| 보더 | none |
| 텍스트 | `oklch(1 0 0)` (흰색) |
| 텍스트 내용 | "네이버로 계속하기" |
| 아이콘 | 네이버 N 로고 SVG (흰색) |
| 호버 | 배경 `oklch(0.58 0.17 145)` |
| 활성 | 배경 `oklch(0.55 0.17 145)`, scale(0.98) |
| 그림자 | none |

### 4.4 Dark Mode 버튼 변형

| 버튼 | Dark Mode 변경사항 |
|------|-------------------|
| Google | 배경 `oklch(0.25 0 0)`, 보더 `oklch(1 0 0 / 15%)`, 텍스트 `oklch(0.92 0 0)` |
| Naver | 변경 없음 (브랜드 색상 유지) |

## 6. 에러 상태

### 6.1 인라인 에러 메시지

OAuth 실패 시 버튼 아래에 표시:

```
┌─────────────────────────────────┐
│  🔵 Google로 계속하기           │
└─────────────────────────────────┘
  ⚠️ 로그인에 실패했습니다.
     다시 시도해주세요.
┌─────────────────────────────────┐
│  🟢 네이버로 계속하기           │
└─────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 색상 | `--destructive` |
| 폰트 | 13px, weight 400 |
| 아이콘 | `AlertCircle` (Lucide), 14px |
| 표시 애니메이션 | fadeIn `--duration-normal` (200ms) |
| 간격 | 위 8px, 아래 12px |

### 6.2 로딩 상태

| 속성 | 값 |
|------|-----|
| 버튼 상태 | disabled, opacity 0.7 |
| 아이콘 교체 | 로고 → 스피너 (spin 애니메이션) |
| 텍스트 | "로그인 중..." |

## 7. 리다이렉트 동작

| 진입 경로 | 로그인 후 이동 |
|----------|---------------|
| 헤더 로그인 버튼 (홈) | 홈으로 복귀 |
| 헤더 로그인 버튼 (랭킹) | 랭킹으로 복귀 |
| 클리어 모달에서 랭킹 등록 | 랭킹 페이지 |
| 직접 URL 접근 | 홈으로 이동 |

---

## 8. 반응형

| 브레이크포인트 | 변경사항 |
|---------------|---------|
| < 375px | 버튼 max-width: 280px, 로고 56px |
| 375px ~ 767px | 기본 (위 와이어프레임) |
| ≥ 768px | 로그인 카드 UI (max-w: 400px, 카드 배경 + 그림자) |
| ≥ 1024px | 좌측 일러스트 + 우측 로그인 폼 (2컬럼) |
