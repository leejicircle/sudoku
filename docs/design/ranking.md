## 3. 난이도 필터 탭

```
┌────────┬────────┬────────┬────────┐
│  쉬움  │  보통  │ 어려움 │ 전문가 │
└────────┴────────┴────────┴────────┘
  ═════                               ← 활성 인디케이터 (2px)
```

### 3.1 탭 스타일

| 속성 | 값 |
|------|-----|
| 레이아웃 | flex, 균등 분할 (flex-1 × 4) |
| 높이 | 44px (터치 타겟) |
| 폰트 | 14px (`--text-caption`), weight 500 |
| 활성 탭 텍스트 | `--foreground`, weight 600 |
| 비활성 탭 텍스트 | `--muted-foreground` |
| 활성 인디케이터 | 하단 2px, 난이도 색상 (`--difficulty-*`) |
| 배경 | transparent |
| 패딩 좌우 | 0 (균등 분할이므로) |
| 전체 하단 보더 | 1px solid `--border` |
| 트랜지션 | 인디케이터 슬라이드, `--duration-normal` (200ms) |

## 4. 상위 3위 포디움 (Podium)

### 4.1 레이아웃

```
         ┌─────┐
         │  1  │ ← 100px 높이
    ┌────┤     ├────┐
    │ 2  │     │ 3  │ ← 80px 높이
    └────┴─────┴────┘
```

### 4.2 포디움 명세

| 속성 | 값 |
|------|-----|
| 전체 높이 | 180px |
| 배경 | `--card` |
| 모서리 | `--radius-lg` (10px) |
| 패딩 | 20px 16px |
| 마진 | 16px (좌우) |

| 순위 | 아바타 크기 | 이름 폰트 | 시간 폰트 | 메달 |
|------|-----------|----------|----------|------|
| 1위 | 56px | 16px bold | 14px mono | 🥇 금색 (`oklch(0.80 0.15 85)`) |
| 2위 | 44px | 14px medium | 12px mono | 🥈 은색 (`oklch(0.75 0.02 250)`) |
| 3위 | 44px | 14px medium | 12px mono | 🥉 동색 (`oklch(0.65 0.10 55)`) |

### 4.3 아바타

| 속성 | 값 |
|------|-----|
| 형태 | 원형 (`--radius-full`) |
| 배경 (이미지 없을 때) | `--muted` |
| 보더 | 2px solid 메달 색상 |
| 이니셜 폰트 | `--font-geist-sans`, bold |
